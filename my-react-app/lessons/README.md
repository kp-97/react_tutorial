# Lesson Guide

Follow the numbered folders in order. In each lesson: read the explanation, predict
the example output, import `Example.jsx` into the playground, complete the exercise,
and verify with `npm run lint` and `npm run build`.

An example can replace the playground temporarily:

```jsx
import Example from '../../lessons/01-jsx/Example.jsx'
export default Example
```

Adjust the relative path based on the importing file. After lessons 1-10, build a
counter with an editable step. After lessons 11-18, complete the capstone in the
root README.
