# Lesson Guide

Follow the numbered folders in order. In each lesson: read the explanation, predict
the example output, import `Example.tsx` into the playground, complete the exercise,
and verify with `npm run typecheck`, `npm run lint`, and `npm run build`.

An example can replace the playground temporarily:

```tsx
import Example from '../lessons/01-jsx/Example'
export default Example
```

This path is correct when used in `src/App.tsx`. Change `01-jsx` to select another
lesson. Each example is a component for the shared Vite playground, not a separate
application. After lessons 1-10, build a
counter with an editable step. After lessons 11-18, complete the capstone in the
root README.
